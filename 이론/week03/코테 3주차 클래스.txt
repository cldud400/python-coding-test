클래스 : 사용자가 정의하는 새로운 타입
객체 : 클래스가 메모리에 만들어지면 객체

method : 클래스 내에서 정의 된 함수
함수 : 모든함수


.의 의미
객체.member   ' '.split()     ' '.format() 등등
class.메서드
module.모듈내의함수

_의 의미
._변수 : 비공개
_method : 비공개
for _ in list 해당 원소를 변수로 쓰지 않겠다

__init__
__name__
__del__
.__변수 : 맹글링 : 해당 변수의 이름을 바꿔서 저장(_class이름__변수)


접근이 불가능하면 getter과 setter을 사용하고
getter과 setter이 없다면 접근하지 말자

추상클래스 : 추상 메서드를 하나라도 포함하고 있는 클래스를 추상 클래스

모듈 : .py파일
패키지 : 모듈을 모아둔 폴더 (__init__.py 파일이 있으면 패키지 취급)
